# Getting Started with SkyMapper

Welcome to the official documentation for the SkyMapper project. 
This guide will walk you through the process of building a low-cost, fully autonomous 3D reconstruction and targeted remote monitoring system using off-the-shelf commercial UAVs and smartphones.

## 🛠️ System Overview
The SkyMapper system is designed with a closed-loop architecture that seamlessly integrates hardware and software components to achieve 7-12cm reconstruction accuracy in unknown environments. The documentation is divided into two main sections:

### 1. [Hardware Setup](hardware.md)
This section covers the physical components required to build the system.
- Commercial drone (UAV) specifications and selection
- Smartphone sensor requirements for Vision-Inertial Odometry (VIO)
- Custom 3D-printed mount assembly instructions

### 2. [Software Deployment](software.md)
This section explains the software stack and execution pipeline.
- Mobile application for real-time tracking and image capture
- Web server configuration for data archiving and online 3D reconstruction
- Autonomous flight controller and view planning modules

Select a topic from the sidebar menu to begin building your own SkyMapper system.
